Github

Descrição

O Github é uma plataforma de gerenciamento de código a qual funciona com base no Git. Se trata de uma ferramenta a qual facilita que outros usuários acessam um determinado código fonte, desde que estes possuam permissões para visualizá-los.

Por se basear na ferramenta Git, a plataforma possui a capacidade de facilitar o controle de versionamento de determinado código fonte, permitindo que os mantedores deste visualizem as alterações realizadas em cada artefato, o que neste caso é um commit.

Justificativa de uso

O desenvolvimento de software tende a ser uma atividade extremamente complexa, a implementação de novas funcionalidades em um sistema pode facilmente causar comportamentos indesejados, sendo assim extremamente interessante a utilização de uma ferramenta a qual permita a visualização de cada alteração realizada no código fonte de um sistema.

Outro ponto em questão, é que o desenvolvimento de projetos na Fabrica de Software é um trabalho realizado em grupos e gerenciar o versionamento de código entre várias pessoas pode causar diversos problemas como mais de um desenvolvedor realizando alterações no mesmo local, junção de funcionalidades desenvolvidas ao mesmo tempo por pessoas diferentes e muitas outras. Estes problemas se tornaram um dos grandes problemas durante o desenvolvimento do Linux, por Linux Torvalds. Buscando uma solução, ele desenvolveu o Git o qual permite o gerenciamento de versões de código descentralizado o qual acompanha todas as modificações realizadas em um projeto e possui a capacidade de mesclar códigos modificações realizadas por um ou mais usuários, ao mesmo tempo que em caso de conflitos possui a capacidade de indicar exatamente onde o problema ocorreu e pedir que escolhemos qual parte do código deverá permanecer, e qual será removida.

Embora o Git, disponibilizada todas essas funcionalidades a ferramenta é utilizada localmente no computador do desenvolvedor. Sendo assim necessário a utilização de um repositório remoto (um projeto com git em um outro computador) o qual irá armazenar todas as alterações realizada no projeto. Para isso, existem diversas ferramentas como Gitlab, Github, etc...

Dentre as plataformas de controle de versão existentes, foi-se escolhido a utilização do Github devido a necessidade de trabalhar em um projeto como um Time. Funcionalidade a qual é disponibilizada gratuitamente no Github enquanto em outras plataformas é necessário que seja efetuado o pagamento de uma determinada quantia.
